# 学员首页

## 入口区

- 查看当前排行榜：进入飞书多维表格仪表盘，查看累计积分、周期表现和最新快照。
- 查看作业要求：阅读本期作业主题、提交格式和截止时间。
- 查看历史记录：回顾自己已提交的作业、最近得分和状态变化。

## 提交说明

1. 在训练营飞书群内发送 PDF 或 DOCX 文件。
2. 如有作业标签要求，请按当期说明填写。
3. 系统会自动抓取文档并完成评分。

## 结果说明

- 排行榜展示累计积分结果。
- 同一周期多次提交时，系统按最高有效分结算。
- 如提交内容无效或文档解析失败，会在结果区显示对应状态。

## 常见问题

- 为什么我没有上榜？
  可能是当前周期尚未形成有效成绩，或你的账号被运营侧设置为不上榜成员。
- 为什么头像和昵称与群里一致？
  看板会优先使用飞书群成员资料进行同步展示。
