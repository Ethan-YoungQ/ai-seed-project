# 运营首页

## 工作入口

- 进入运营视图：查看成员资料、提交记录、周期结果、预警和快照。
- 进入排行榜仪表盘：确认学员侧展示是否正常。
- 查看系统状态：检查飞书接入、Base 表配置和最近一次入站处理结果。

## 日常操作

1. 核对成员资料，确认学员昵称和头像同步正常。
2. 检查异常提交、解析失败和待处理状态。
3. 对非学员或运营人员执行不上榜设置。
4. 预览并发送排行榜播报或提交汇总。

## 排查提示

- 若成员头像或昵称未更新，先检查飞书成员资料同步是否成功。
- 若看板未显示最新结果，先检查最近一次入站事件和 Base 同步状态。
- 若需要调整入口链接，请更新一期入口配置并重新执行引导脚本。

## 权限说明

- 学员只进入学员首页和公开看板。
- 运营与管理员可进入完整运营视图并执行管理动作。
