#### B. 学员流程（测试学员操作，30 min）

| # | 测试项 | 操作步骤 | 预期结果 |

| C.5 | 成员管理 | 通过 API 修改成员角色 | 角色变更生效 |
| C.6 | 开期操作 | 发送 /开期 2 命令 | Period 2 开启，进入评分期 |

## 风险与应急

| 风险 | 概率 | 影响 | 应急措施 |
|------|------|------|----------|
| GLM API 限流 (429) | 中 | 评分延迟 | 已配置指数退避 + 3 次重试 |
| 飞书 WebSocket 断连 | 低 | 事件丢失 | systemd 自动重启 + 重连机制 |

| ICP 端口封锁 | 中 | 外网不可达 | 使用飞书 H5 代理（Path 1） |
| 测试发现严重 bug | 中 | 延期 | Phase 2 整天用于修复 |
