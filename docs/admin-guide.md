# 辉瑞 HBU AI 训练营 — 管理员操作手册

> 14 名学员 / 12 个周期 / 5 个维度 / 6 个段位
>
> 所有操作均在飞书群「HBU奇点玩家」内完成，无需任何技术背景。

---

## 一、系统是什么

一套自动追踪学员学习表现的评分系统。学员在飞书群打卡、答题、互动，系统自动计分并生成排行榜。管理员只需在群里点几下按钮。

---

## 二、你能做什么

| 操作 | 怎么触发 | 做什么 |
|------|---------|--------|
| 管理面板 | 群里发 **管理** | 开新周期、开评分窗口、触发毕业结算、刷新状态 |
| 审核评分 | Bot 自动推卡片到群 | 点 ✅ 批准 或 ❌ 驳回 |
| 手动调分 | 群里发 **调分** | 选学员 → 选评分项 → 填分值 → 确认 |
| 成员管理 | 群里发 **成员** | 修改角色、隐藏/显示排行榜 |
| 看排行榜 | 浏览器 或 飞书工作台 | 查看学员积分、段位、详情 |

---

## 三、日常操作步骤

### 1. 审核 AI 评分（每天检查）

学员提交文字类作业后，AI 会自动打分，然后推一张审核卡片到群里。

```
群内会出现这样的卡片：

┌─────────────────────────────────┐
│  待审核：张三 提交了 K3 知识总结   │
│  AI 评分：3 分                    │
│  AI 评语：内容详实，符合要求        │
│                                   │
│     [ ✅ 批准 ]    [ ❌ 驳回 ]     │
└─────────────────────────────────┘
```

**操作：** 看一眼 AI 评语，觉得合理就点 ✅，不合理就点 ❌。

> 小贴士：一般 AI 判断都比较准，快速浏览即可。

---

### 2. 开启新周期（每个周期结束时）

**第一步：** 在群里发送 `管理`

**第二步：** Bot 回复管理面板卡片

```
┌─────────────────────────────────┐
│        管理员控制面板              │
│                                   │
│  当前周期：Period 3                │
│  当前窗口：W2                      │
│                                   │
│  ── 开启新周期 ──                  │
│  选择周期：[ 1 ▼ ]  [ 开启 ]      │
│                                   │
│  ── 开启评分窗口 ──                │
│  选择窗口：[ W1 ▼ ]  [ 开启 ]     │
│                                   │
│  ── 其他 ──                        │
│  [ 触发毕业结算 ]  [ 刷新状态 ]    │
└─────────────────────────────────┘
```

**第三步：** 选择下一个周期编号（1-12），点「开启」

> 注意：Period 1 是破冰期，不计分。

---

### 3. 手动调分（偶尔需要）

**第一步：** 在群里发送 `调分`

**第二步：** Bot 回复调分卡片

**第三步：** 依次选择：
- 选哪个学员
- 选哪个评分项
- 填写分值变化（正数加分，负数扣分）
- 点确认

---

### 4. 成员管理（偶尔需要）

**第一步：** 在群里发送 `成员`

**第二步：** Bot 回复成员管理卡片

**第三步：** 可以：
- 修改某人的角色
- 从排行榜隐藏/显示某人

---

### 5. 查看排行榜

**方式一：** 浏览器打开

```
http://114.215.170.79:3000/dashboard/
```

**方式二：** 飞书工作台 → 点击「辉瑞 HBU AI 训练营评估系统」

排行榜上可以看到：

## 四、评分规则速查

### 五大维度

| 代号 | 维度 | 一句话解释 |
|------|------|-----------|
| K | 知识 | 学了多少 |
| H | 实操 | 动手做了多少 |
| C | 创新 | 有没有创意 |
| S | 社交 | 群里活不活跃 |
| G | 增长 | 比上次进步了多少 |

**自动计分（9 个）** — 学员操作后立刻得分，无需管理员介入

|------|------|-------|---------|

|------|------|-------|---------|---------|

| C1 | 创新 | 4 | 8 | 写 20 字以上总结 |

### 段位晋升

```
M1 入门 → M2 初级 → M3 中级 → M4 高级 → M5 专家 → M6 大师
```

- 每 **2 个周期** 自动结算一次，根据积分决定是否晋升
- 连续未晋升会有保护机制（降低门槛）
- 全勤学员在最终窗口有额外优待

```
Period 1     → 破冰期（不计分）
Period 2-3   → 窗口 W1 → 结算 → 更新段位
Period 4-5   → 窗口 W2 → 结算 → 更新段位

Period 10-11 → 窗口 W5 → 结算 → 更新段位
Period 12    → FINAL  → 毕业结算 → 最终段位
```

## 七、常见问题

**Q：审核卡片太多，能不能批量通过？**
> 目前需要逐条审核。好消息是大部分 AI 判断都很准，一般快速点批准就行。

**Q：学员说分数不对怎么办？**
> 群里发 `调分`，选择学员和评分项，填写正确的分值变化。

**Q：想临时把某人从排行榜上隐藏？**
> 群里发 `成员`，找到该学员，点隐藏。

**Q：排行榜打不开了？**
> 先等几分钟再试。如果持续打不开，联系技术支持。

**Q：开错周期了怎么办？**
> 操作是幂等的（重复操作不会出错）。如果编号选错了，联系技术支持回滚。

**Q：破冰期（Period 1）需要做什么？**
> 什么都不用做。破冰期不计分，学员熟悉系统即可。

**Q：毕业结算后还能调分吗？**
> 可以。用 `调分` 功能随时修改。
