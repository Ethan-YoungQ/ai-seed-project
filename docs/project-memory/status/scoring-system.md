## 自动捕获架构

```
学员消息 → WS → message-classifier.ts (返回多项)
  → K1 始终包含（任何活动=签到）
  → 内容匹配: URL→G2, 文件→H1, 图片→H2, 视频关键词→H3/G1
  → 长文本启发式: prompt→C3, 纠错→K4, 创意→C1, 默认→K3
  → Ingestor 10步管道写入 DB
  → Bot 发确认消息（最高价值项，5分钟限流）
```
